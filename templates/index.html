Welcome to the REST API

API Dokumentation

Diese REST API erlaubt es, Personen zu verwalten (lesen, hinzufügen, aktualisieren und löschen). Die Daten werden in einer lokalen JSON-Datei gespeichert.

HTTP Methode Pfad Beschreibung Request Body Response
GET /person/ Alle Personen abrufen – JSON-Array aller Personen
POST /person/ Neue Person erstellen {"id": "123", "name": "Anna"} Erstellte Person (Status 201)
GET /person/<id> Eine bestimmte Person abrufen – Person als JSON oder Fehler
PUT /person/<id> Person aktualisieren {"id": "123", "name": "Max"} Aktualisierte Person oder Fehler
DELETE /person/<id> Person löschen – Gelöschte Person oder Fehler

Hinweis: Die Daten werden in der Datei data.json gespeichert und sind daher nicht persistent bei Server-Neustart (außer du speicherst sie bewusst).
